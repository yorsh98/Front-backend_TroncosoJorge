# Plan De Sprints

- Sprint 0: Preparacion, analisis y estructura base.
- Sprint 1: Backend Laravel API, Docker y MySQL.
- Sprint 2: Autenticacion, roles, permisos y seeder superusuario.
- Sprint 3: Modelo de datos principal.
- Sprint 4: Configuracion IA desde backend y panel superusuario.
- Sprint 5: Analisis de CV PDF, DOCX, DOC, LibreOffice, regex, Ollama y OpenAI.
- Sprint 6: API de personas, perfil laboral y CV ciego.
- Sprint 7: API de empresas y vitrina de talentos.
- Sprint 8: API administracion Departamento de Empleo.
- Sprint 9: Swagger/OpenAPI y documentacion backend.
- Sprint 10: Frontend Laravel React Starter Kit.
- Sprint 11: Landing, auth y diseno Providencia.
- Sprint 12: Frontend persona, CV y modal de datos personales.
- Sprint 13: Frontend empresa y vitrina de talentos.
- Sprint 14: Frontend admin y superusuario.
- Sprint 15: Accesibilidad universal.
- Sprint 16: Seguridad, privacidad, auditoria y datos reales.
- Sprint 17: Reportes y exportacion Excel.
- Sprint 18: Tests, seeders y datos demo.
- Sprint 19: Documentacion final para profesores.
- Sprint 20: ZIP final y validacion general.

Cada sprint debe crear o modificar archivos reales, ejecutar comandos cuando el entorno lo permita, corregir errores y documentar resultados antes de avanzar.
